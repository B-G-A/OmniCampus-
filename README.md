# OmniCampus

OmniCampus is an AI-assisted web platform for academic support and campus collaboration.

## Features

- AI Syllabus Assistant
- Student Dashboard
- Teacher Dashboard
- Academic Resource Management
- Campus Club & Collaboration Hub
- Centralized Learning Support

## Tech Stack

- React.js
- JavaScript
- CSS
- Future Backend Integration (Python)

## Project Goal

To create a unified digital ecosystem that improves academic support, communication, and collaboration among students, faculty, and campus organizations.
